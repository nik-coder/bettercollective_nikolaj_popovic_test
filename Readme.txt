Post from Category plugin fork

1. This is a public repository on Github
2. install plugin by installing from WP admin add new plugin or by uploading plugin folder into wp-content/plugins/ directory
3. Activate the plugin

This plugin has just  a small change in this fork which enables post when selected post category in the widget to show posts by date ASC or DESC, all other credits goes to original developer of the plugin. This is just a fork.

From WP Admin click on Appearance->Widgets, then add new widget area or add posts from category widget in an existing widget area. Edit fields as per shown in the widget.
